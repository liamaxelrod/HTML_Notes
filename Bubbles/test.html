The <head> Element Summary

The <head> of an HTML document contains metadata — information about the page that's not displayed directly. It's placed between <html> and <body>.

Common Elements in <head>:

<title> – Sets the page title (shown in browser tab, bookmarks, and used by search engines). Required. Important for SEO.

<style> – Adds internal CSS directly to the page. Use for small, page-specific styles.

<link> – Links to external files, commonly CSS stylesheets. Example: <link rel="stylesheet" href="style.css">

<meta> – Defines metadata such as:

<meta charset="UTF-8"> – Character set definition

<meta name="description" content="..."> – Description for search engines

<meta name="keywords" content="..."> – Keywords (not widely used today)

<meta name="author" content="..."> – Author information

<meta http-equiv="refresh" content="30"> – Page refresh setting

<meta name="viewport" content="width=device-width, initial-scale=1.0"> – Sets layout scaling and responsiveness on all devices
